# List Methods

| 🔢 **Method**         | 📝 **Description**                                                          |
| --------------------- | --------------------------------------------------------------------------- |
| `.append(x)`          | ➕ Adds item `x` to the **end** of the list.                                |
| `.extend(iterable)`   | 🔗 Adds **all elements** from another iterable (like a list) to the end.    |
| `.insert(i, x)`       | 🎯 Inserts item `x` at **index `i`**.                                       |
| `.remove(x)`          | ❌ Removes the **first occurrence** of item `x`.                            |
| `.pop(i)`             | 🧹 Removes and returns the element at **index `i`**. (Default is last item) |
| `.clear()`            | 🧼 Clears all elements from the list — makes it empty.                      |
| `.index(x)`           | 🔍 Returns the **index** of the first occurrence of item `x`.               |
| `.count(x)`           | 🔢 Counts how many times item `x` appears in the list.                      |
| `.sort()`             | 🔼 Sorts the list in **ascending order** (low to high).                     |
| `.sort(reverse=True)` | 🔽 Sorts the list in **descending order** (high to low).                    |
| `.reverse()`          | 🔁 Reverses the **current order** of the list.                              |
| `.copy()`             | 📋 Creates a **shallow copy** of the list (not linked to the original).     |
