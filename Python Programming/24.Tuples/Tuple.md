🔹 What Are Tuples in Python?
A tuple is a built-in data type in Python used to store a collection of items, just like a list — but with one major difference:

🔒 Tuples are immutable — meaning you cannot change, add, or remove elements after the tuple is created.

| Feature               | Description                                                      |
| --------------------- | ---------------------------------------------------------------- |
| **Immutable**         | You **cannot** modify the contents after creation.               |
| **Ordered**           | Items have a **defined order** and can be accessed by index.     |
| **Allows duplicates** | You can have repeated values.                                    |
| **Faster than lists** | Because they are immutable, they are slightly faster than lists. |
